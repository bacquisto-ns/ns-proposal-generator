# Frontend Developer Skill

This skill focuses on delivering "premium" web experiences that wow users through aesthetics, responsiveness, and polished interactions.

## Core Principles
1. **Visual Excellence**: Use high-quality color palettes, modern typography, and smooth transitions.
2. **Responsive by Default**: Ensure layouts work flawlessly across mobile, tablet, and desktop.
3. **SEO & Performance**: Implement semantic HTML, meta tags, and optimize asset loading.
4. **Interactive Logic**: Use clean JavaScript for dynamic UI updates without compromising performance.

## Implementation Checklist
*   [ ] **Design System**: Use variables for colors, spacing, and typography.
*   [ ] **Aesthetics**: Apply subtle gradients, glassmorphism, or micro-animations where appropriate.
*   [ ] **Semantics**: Use `<header>`, `<main>`, `<footer>`, and proper heading hierarchy.
*   [ ] **Accessibility**: Ensure high contrast ratios and keyboard navigability.
*   [ ] **Validation**: Check form inputs on the client side before submission.
